Topic: Hbase
Architecture: Single Node Cluster
Programming Language: JAVA
Problem: Store name and lastname of a student and a proffesor inside two columns (one for each one)
	 The table has to be created first and deleted after the program runs and shows the data.

Pre-requisites:
- Having an IDE installed
- Having Hbase, Hadoop and Zookeeper running. 

How to run it:
- Create a new project
- Copy the HBaseUniversity.java inside the project
- Import the required jars (they are provided in the requiredLibraries folder)
- Run the main method of the HBaseUniversity.java
